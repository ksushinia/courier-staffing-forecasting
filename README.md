# Прогнозирование нехватки курьеров в дарксторах

Цель проекта -- предсказать, сколько сотрудников потребуется даркстору на неделю вперед, чтобы успешно обработать поток заказов без задержек.

В распоряжении были исторические данные по 65+ дарксторам:
- Фактическое количество заказов и сотрудников за прошлые периоды.
- Процент опозданий курьеров.
- Данные о маркетинговых затратах и оттоке персонала.
- Город и время работы даркстора.

**Что было сделано:**
- **Feature Engineering:** Созданы лаги для ключевых признаков, рассчитаны скользящие средние для сглаживания недельных колебаний.
- **Обработка данных:** Работа с пропусками, кодирование категориальных признаков (города).
- **Моделирование:** Использован градиентный бустинг (CatBoost), так как на данной задаче с данными табличного типа он показал себя лучше всего.
- **Валидация:** Проверка модели проводилась с учетом временной структуры данных.

**Результаты**
- Модель вошла в топ рейтинга на соревновании (место 8, score: 0.17863720) (https://www.kaggle.com/competitions/intern-regression-courier-deficit-challenge)
